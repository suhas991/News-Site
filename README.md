- ## News Site

Collections of latest news by different categories using NEWS_API

### Getting Started

1. Clone this repository:

   ```bash
   git clone https://github.com/suhas991/News-Site.git
   ```

2. Install any required dependencies (if applicable):

   ```bash
   npm install
   ```

3. Run the game:


   ```bash
   npm run dev
   ```
